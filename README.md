## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de unir a estruturação semântica do **HTML**, a estilização personalizada do **CSS** e a agilidade e responsividade do **Bootstrap**.

### Tecnologias Utilizadas

O trio de tecnologias escolhido garante uma estrutura sólida, design atraente e total adaptação a qualquer tamanho de tela:

* **HTML:** Responsável pela arquitetura e conteúdo semântico da página.
* **CSS:** Utilizado para estilizações personalizadas, transições suaves e ajustes finos que o framework não cobre por padrão.
* **Bootstrap:** Framework utilitário usado para acelerar o desenvolvimento do layout, criação do sistema de grid (responsivo) e componentes dinâmicos (como menus e carrosséis).

---

## Funcionalidades e Destaques

* **Design 100% Responsivo:** Perfeita visualização em smartphones, tablets e desktops graças ao sistema de grid do Bootstrap.
* **Estilização Híbrida:** O poder dos componentes prontos do Bootstrap combinado com a identidade visual exclusiva feita em CSS puro.
* **Acessibilidade:** Uso de tags semânticas do HTML (`<header>`, `<footer>`, `<section>`) para melhor indexação no Google  e leitores de tela.
